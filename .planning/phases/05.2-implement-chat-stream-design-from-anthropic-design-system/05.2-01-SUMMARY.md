## Decisions Made

- **Test file location moved from `src/lib/chat-format.test.ts` (plan) to `tests/lib/chat-format.test.ts` (convention)** — the plan acceptance criteria asked for the src/-co-located path, but `vitest.config.ts` line 17 sets `include: ['tests/**/*.test.{ts,tsx}']`. A co-located test would never be picked up by `npm test` or CI. Honored the project mirror-pattern convention (every existing test under `tests/lib/`). Documented as Rule 3 deviation.
- **`@/lib/...` alias imports in test file** — matches the established `tests/lib/cost.test.ts` pattern; vitest config has `resolve.alias['@'] = ./src`.
- **No memoization** — plan explicitly forbids `useMemo` wrappers; CHAT-10 caps conversation at 30 turns and the ops are trivial.
- **System-message handling matches RESEARCH §Pattern 2 verbatim** — `if (m.role === 'system') continue` skips system from OUTPUT, but `prev = messages[i-1]` looks at raw array (system as neighbor breaks the same-role run). Test case `[user, system, user]` → both users get `'only'`.
